
# Laptop Warranty & Replacement Dashboard

## Overview

Power BI solution used for proactive replacement planning and warranty monitoring.

## Features

- Warranty expiry tracking
- Replacement forecasting
- Device age analysis
- Vendor reporting
- Budget planning

## Skills Demonstrated

- Asset Lifecycle Management
- DAX
- Power Query
- Forecasting
