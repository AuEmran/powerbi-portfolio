# LGA Case Dashboard

## Business Problem

Management required visibility of case activity geographically across NSW.

## Solution

Developed an Azure Maps dashboard using ABS LGA boundaries and Power BI.

## Key Features

- Interactive LGA map
- Dynamic filtering
- Drillthrough capability
- Case volume heat mapping
- Geographic trend analysis

## Technology

- Power BI
- DAX
- Power Query
- Azure Maps
- GeoJSON
